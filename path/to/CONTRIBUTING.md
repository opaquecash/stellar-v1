# Contributing

We welcome contributions from the community! Please follow these guidelines:

- **Update the CHANGELOG**: After each pull request or merge, update the `CHANGELOG.md` file in the `Unreleased` section with a summary of your changes, following the [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) format.
- **Follow SemVer**: Ensure version numbers are incremented according to [Semantic Versioning](https://semver.org/spec/v2.0.0.html) guidelines.
- **Keep it concise**: Focus on what changed and why, avoiding excessive detail.

Your contributions help keep our project transparent and easy to follow.
