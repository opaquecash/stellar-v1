# Contributing

We welcome contributions from the community! Please follow these guidelines:

- **Update the CHANGELOG**: After each merge to `main`, add an entry to `CHANGELOG.md` in the `Unreleased` section for your changes. If you're releasing a new version, also add an entry for that version.
- **Follow SemVer**: Ensure version numbers follow Semantic Versioning.
- **Keep it concise**: Focus on what changed and why.
- **Submit a PR**: Open a pull request with clear descriptions of your changes.

For detailed instructions, see our [README](README.md).
